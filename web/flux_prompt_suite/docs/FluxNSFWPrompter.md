# Flux NSFW Prompter

Produce concise positive/negative prompts tuned for Flux from high-level controls (subject, style, camera, lighting, safety).

## Inputs
- `subject` (STRING): Short subject description.
- `style`, `camera`, `lighting` (COMBO): Aesthetic knobs.
- `spice` (FLOAT 0..1): Adds accent tokens as it increases.
- `safety_level` (COMBO): Adjusts negative tokens for SFW/NSFW sensitivity.
- `use_grok` (BOOLEAN): Optional external upgrade via route/JS.
- Optional: `grok_api_key`, `guidance_tags`, `forbidden_tags`, `prompt_override`, `use_canvas_ui`.

## Outputs
- `positive` (STRING), `negative` (STRING)

## Notes
- Deterministic local templates; no explicit content is generated by the code.
- `IS_CHANGED` covers widget changes; prompts are recomputed as needed.
